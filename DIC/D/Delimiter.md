Delimiter는 **구분자** 라는 뜻을 가집니다.

일반적으로 문자열을 나눌 때, 기준점을 의미할 때 사용됩니다.

ex)

```
const string = '이것은,예제,입니다.'
const Delimiter = ','
string.split(Delimiter) // [이것은, 예제, 입니다.]
```
